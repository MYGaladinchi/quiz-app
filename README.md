# Quiz Night

A simple browser-based trivia quiz built with plain HTML, CSS, and JavaScript.

## What it does

- 10 general knowledge questions, one at a time
- Instant feedback on each answer (correct answer highlighted)
- Live score tracking as you go
- Final results screen with a message based on your score

## Run it locally

No build tools or dependencies needed. Just open `index.html` in your browser.

## Project structure

```
quiz-app/
├── index.html   # page structure
├── style.css    # chalkboard-themed styling
└── script.js    # quiz logic and state
```
